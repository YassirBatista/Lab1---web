Glosario – Diseño Web Responsive

Conceptos básicos para diseñar páginas web que se adapten a cualquier tipo de pantalla.

'

Media Queries

Las Media Queries son una característica de CSS que permite aplicar estilos condicionales según las características del dispositivo o la pantalla, como su ancho, alto, resolución u orientación. Se escriben con la regla @media y son la base del diseño responsive, ya que permiten adaptar el layout y la tipografía a móviles, tablets y escritorios sin necesidad de múltiples archivos CSS.

󱗝

Flexbox

El Módulo de Caja Flexible (Flexbox) es un sistema de diseño unidimensional de CSS que organiza elementos en una sola fila o columna. Facilita la distribución del espacio, la alineación y el reordenamiento de elementos dentro de un contenedor, sin importar su tamaño. Es especialmente útil para construir barras de navegación, tarjetas y componentes que deben adaptarse de forma fluida al espacio disponible.

󰓪

Viewport

El viewport representa el área visible de una página web dentro del navegador, es decir, la porción de contenido que el usuario puede ver sin hacer scroll. En dispositivos móviles, su tamaño varía considerablemente respecto al escritorio, por lo que la etiqueta <meta name="viewport"> es esencial para indicarle al navegador cómo escalar y ajustar el contenido según el ancho real del dispositivo.



Mobile First

Mobile First es una metodología de diseño web que prioriza la versión móvil como punto de partida, y luego escala el diseño hacia pantallas más grandes mediante Media Queries. Este enfoque mejora el rendimiento en dispositivos con recursos limitados, obliga a simplificar la interfaz a lo esencial y está alineado con las buenas prácticas actuales de SEO, ya que los motores de búsqueda indexan primero la versión móvil de los sitios.

󰝘

CSS Grid Layout

A diferencia de Flexbox, CSS Grid es un sistema de diseño bidimensional que permite controlar simultáneamente filas y columnas dentro de un contenedor. Es ideal para construir estructuras de página completas, como layouts con cabecera, barra lateral y área de contenido. Con propiedades como grid-template-columns y grid-template-rows, se pueden crear diseños complejos y responsivos con muy pocas líneas de código.
